 Uma API que faz uma tabela, podendo ser customizada, a sua preferência, tendo que trocar o "ID" o "nome" e a "idade", 
também sendo possível vê-la em formato de https.
 Este foi um projeto de back end requisitado pelo meu professor, foi bem divertido e meio desafiador fazê-lo, mas gostei bastante
do resultado final e de como este projeto pode mudar a minha visão sobre as APIs e os próprios códigos em JavaScript.
